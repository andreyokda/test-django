# Тестовое задание: Слайдер на Django + Bootstrap 5

## Описание проекта
Сайт с адаптивным слайдером фотографий, управляемым через админку Django.

## Технологии
- Python 3.12
- Django 5.2
- MySQL
- Bootstrap 5
- Slick Slider (синхронизация)
- Fancybox (полноэкранная галерея)
- django-filer (загрузка изображений)
- django-admin-sortable2 (сортировка)

## Функционал
- Адаптивная верстка (десктоп/планшет/мобилка)
- Бургер-меню на мобильных
- Слайдер с синхронизацией большого фото и превью
- Полноэкранный просмотр по клику
- Админка с drag&drop сортировкой слайдов
- Загрузка фото через django-filer
